THE LOVELLS WEBSITE
===================

HOW TO VIEW
1. Unzip the folder.
2. Open index.html in Chrome, Safari, Edge or Firefox.
3. The website works as a static local site, no server or installation is required.

HOW TO REPLACE THE PLACEHOLDER IMAGES
- Add your own JPG or WebP images to assets/images.
- In index.html, update each gallery tile's image filename.
- In each file inside the galleries folder, replace the image src values with your own filenames.
- Keep image alt text descriptive, for example: "Becky and Stuart walking through the confetti after the ceremony".

SUGGESTED IMAGE SIZES
- Homepage covers: 1600 x 1200 pixels or larger.
- Gallery images: at least 1600 pixels wide.
- WebP is recommended for smaller file sizes and faster loading.

PRIVACY NOTE
This static version has no password protection. Before publishing personal or newborn photography online, add hosting-level password protection or use a platform that supports private galleries.
